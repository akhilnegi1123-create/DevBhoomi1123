# 🏔️ DevBhoomi Travels — Backend Server

## ▶️ Chalane ke liye sirf 2 commands:

```bash
cd backend
npm install
npm start
```

## 🌐 URLs (server start hone ke baad):

| Page | URL |
|------|-----|
| 🏠 Website | http://localhost:5000 |
| 🔐 Admin Panel | http://localhost:5000/admin.html |
| 🏨 Hotels | http://localhost:5000/Hotel.html |
| 🚗 Vehicles | http://localhost:5000/Vehicle.html |
| ❤️ API Health | http://localhost:5000/api/health |
